# Kubernetes Configs – Learning Basics

## 📌 Purpose
This repository is created for **learning Kubernetes fundamentals**. It contains example manifests to practice with ConfigMaps, Secrets, Deployments, and Services.

## 📂 Folder Structure
```
examples/
└── python-web-app/
├── configmaps/
│   └── app-config.yaml
├── secrets/
│   └── app-secret.yaml
├── deployments/
│   └── app-deployment.yaml
└── services/
└── app-service.yaml
```

## ⚠️ Important Note
- Secrets in this repository contain **dummy values** for demonstration purposes.  
- Replace them with valid credentials before using in any real environment.  
- These manifests are intended for **learning and practice only**, not production use.
